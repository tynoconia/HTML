HTML

HTML is a Hyper Text Markup Language. It is used to create webpages on the internet. Websites are connected using hyper links/text.

HTML uses tags, which divides webpages into elements. These carry instructions for what to do with the text.

The browser is important too, because the code is telling the browser what to show on the screen.

This language is very simple to use and learn.

HTML Uses

HTML is used for almost every single website! It is used to format and create the text. Without HTML, who knows what websites like:

  • Schoology
  • Youtube
  • Google

would look like! Without HTML, webpages would look and be vastly different!
